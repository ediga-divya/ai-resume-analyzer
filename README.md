# AI Resume Analyzer

## Overview

AI Resume Analyzer is a Python and Streamlit application that evaluates resume text for ATS keyword alignment, technical skill relevance, and role-based matching. The project helps candidates understand how well their resume aligns with Software Engineer, Data Engineer, and AI/ML Engineer roles.

## Features

* Resume keyword analysis
* Skill extraction
* ATS-friendly scoring
* Role-based matching
* Interactive Streamlit user interface
* Real-time feedback and recommendations

## How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/ediga-divya/ai-resume-analyzer.git
cd ai-resume-analyzer
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the app:

```bash
streamlit run app.py
```
