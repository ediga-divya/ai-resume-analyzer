
# AI Resume Analyzer

## Overview

AI Resume Analyzer is a Python-based application designed to analyze resumes and provide insights using AI-driven text processing techniques. The application helps evaluate resume structure, keyword relevance, and technical skill alignment for software engineering and data engineering roles.

## Features

* Resume keyword analysis
* Skill extraction
* ATS-friendly resume evaluation
* Interactive user interface
* Real-time analysis and scoring

## Tech Stack

* Python
* Streamlit
* Pandas
* NLP/Text Processing

## Project Goals

This project demonstrates:

* AI-driven text analysis
* Backend data processing
* Scalable Python workflows
* Resume optimization concepts
* Interactive application development
